# Visual NDVI

Select and Download tiff files with NDVI bands from Google Earth Engine 

Current Functionality : 

  * Interactive Map.
  * Search for Area.
  * Custom Time Frame.
  * Download the Clipped Image from map to local machine.

Future Scope :

 * Rainfall Data Analysis
 * UI based project
